# Tile Art Licensing

Mahjong tile SVGs from [perthmahjongsoc/mahjong-tiles-svg](https://github.com/perthmahjongsoc/mahjong-tiles-svg).
Licensed under **CC-BY-SA-4.0**.

Files are unmodified; renamed from Unicode codepoint filenames (e.g. 🀇.svg)
to MPSZ-style names (1m.svg, 1p.svg, ..., 5z.svg, ..., 7z.svg) for URL safety.
Mapping:

- 1m–9m ← 萬 (🀇..🀏)
- 1p–9p ← 筒 (🀙..🀡)
- 1s–9s ← 索 (🀐..🀘)
- 1z = East   (🀀)
- 2z = South  (🀁)
- 3z = West   (🀂)
- 4z = North  (🀃)
- 5z = White Dragon (🀆)
- 6z = Green Dragon (🀅)
- 7z = Red Dragon   (🀄)

Because CC-BY-SA-4.0 is share-alike, the bundled tile art carries forward
under the same license. The rest of this project's source remains under
its own terms.
